description: Examples of sales-y phrasing that got toned down to fit the Mustgo "차분·신뢰형" voice

Mustgo의 보이스는 차분하고 신뢰감 있는 톤이며 자극적/단정적/과장된 표현을 피해야 합니다.

**Why:** `MustGo_디자인_프롬프트_v2.md` 와 CLAUDE.md 에 명시된 brand voice. 한국 B2B 사용자는 자기과시적 헤드라인보다 절제된 표현에 더 신뢰를 보냅니다.

**How to apply:** 카피 작성/검토 시 다음 패턴을 피하고, 더 measured 한 표현으로 대체:

| Avoid (sales-y) | Prefer (measured) |
| --- | --- |
| 가장 빠른 답변, ...가 드립니다. | 빠른 회신, ...가 드립니다. |
| ...의 결과를 바꿉니다. | ...의 흐름을 좌우합니다. |
| 최고의 / 최상의 / 최저가 | 최적의 / 적합한 |
| 단언/단정조 ("...할 것입니다") | 약속조 ("...해 드립니다", "책임집니다") |

특히 헤딩 카피에서 "가장 ~한", "최고의" 류 superlative 는 Trust-First 보이스와 충돌합니다. CTA 라벨은 행동 유도성("문의하기", "상담 신청")은 유지하되 자극적 어휘("지금 당장", "놓치지 마세요") 는 사용하지 않습니다.
